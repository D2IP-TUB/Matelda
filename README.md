# ED-Scale

## Installation 

Setup the repo:
```
git clone git@github.com:LUH-DBS/ED-Scale.git
cd ED-Scale
make install
```

Prepare Santos: 
```
make setup-santos
```

Start ED-Scale
```
make run
```

## Utilities

Clean up logs:
```
make clean-logs
```

Clean up logs:
```
make clean-yago
```

Clean up logs:
```
make clean-santos
```

Clean up everything:
```
make clean-all
```

Uninstall:
```
make uninstall
```
